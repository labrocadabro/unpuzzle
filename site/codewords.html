Codewords Helper

Codewords (or sometimes codecrackers) are a type of word puzzle that looks like a crossword without clues. Each cell contains a number that corresponds to a single letter of the alphabet and all 26 letters are used. The goal is to fill the entire grid with valid words.

This tool helps you find word possibilities based on word length and known letters. To use this tool, enter any known letters, separated by numbers representing the number of empty cells between those letter.

For example: If you are looking for a 3 letter word that begins with C and ends with T, enter c1t. If you are looking for a 5 letter word where the first and fourth letters are T, enter t2t1.
